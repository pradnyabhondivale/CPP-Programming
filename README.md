# C++ Practice Repository

## 📌 Overview
This repository contains my C++ programming practice files. It includes various programs written while learning and improving my understanding of C++ concepts, problem-solving skills, and data structures.
The goal of this repository is to strengthen core programming fundamentals and build consistency in coding practice.

---

## 🎯 Purpose

- Strengthen C++ programming fundamentals
- Practice problem-solving skills
- Learn and implement Data Structures and Algorithms (DSA)
- Improve coding consistency
- Maintain a collection of solved programs for revision

---

## 🧠 Topics Covered

This repository includes practice programs on:

- Basics of C++ (syntax, variables, data types)
- Control statements (if-else, switch, loops)
- Functions and recursion
- Arrays and strings
- Pointers
- Object-Oriented Programming (OOP)
  - Classes and Objects
  - Inheritance
  - Polymorphism
  - Encapsulation

## ⚙️ How to Run

### Compile any program:
```bash
g++ filename.cpp -o output
```

### Run the program:

**Windows:**
```bash
output.exe
```

**Linux / Mac:**
```bash
./output
```
